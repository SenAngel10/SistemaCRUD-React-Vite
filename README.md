# React + Vite
Título: Sistema de Inventario
Descripción: Aplicación web para gestión de inventario con funcionalidades basicas. Permite agregar, editar, eliminar y buscar productos por nombre, con filtrado por categorías dinámicas y persistencia en localStorage.
Tecnologías: React, Vite, CSS
Funcionalidades:

CRUD completo de productos
Categorías dinámicas
Búsqueda en tiempo real
Persistencia con localStorage
Diseño responsive
